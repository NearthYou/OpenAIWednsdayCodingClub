# OpenAIWednsdayCodingClub

덕후 대상 일정 플랫폼 프로토타입 저장소입니다.

## Stack

- `frontend`: React + Vite
- `backend`: Node.js + Express

## Current Work

- 브랜치: `codex/page/3`
- 구현 대상: 일정 상세 / AI 요약 페이지

## Run

Node.js 18+가 설치되어 있어야 합니다.

```bash
npm install
npm run dev
npm run server
```

프론트는 기본적으로 `5173`, 백엔드는 `4000` 포트를 사용합니다.
